# Micro-Frontends

### In Vanilla JS
* Setup
```shell
# Create folder structure for products
mkdir vanilla-js
cd vanilla-js
mkdir products
cd products
npm init -y

# Install dependencies
npm install webpack@5.4.0 webpack-cli@4.5.0 webpack-dev-server@3.11.0 html-webpack-plugin@5.1.0 faker@5.1.0
npm install -D @types/faker@5.1.0

# Create container
cd ..
mkdir container
cd container
npm init -y

# Install dependencies
npm install nodemon webpack@5.4.0 webpack-cli@4.5.0 webpack-dev-server@3.11.0 html-webpack-plugin@5.1.0
```
